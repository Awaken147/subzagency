### 1.4 Quality Guidelines

**Domain Authority (DA) Thresholds:**
- **Tier 1 (Premium):** DA 50+ — Always pursue, any relevance
- **Tier 2 (Quality):** DA 30–49 — Pursue if topically relevant (web design, AI, tech, marketing, Sikkim/India business)
- **Tier 3 (Foundational):** DA 15–29 — Acceptable for local citations and niche relevance only
- **Reject:** DA < 15 with no topical relevance, or any site with spam score > 30%

**Relevance Criteria:**
- Primary: Web design, web development, 3D design, AI/ML, automation, digital marketing
- Secondary: Sikkim/Northeast India business, Indian startups, small business resources
- Tertiary: Technology, creative agencies, SaaS, entrepreneurship

**Anchor Text Distribution (Target):**

| Anchor Text Type | Target % | Example |
|---|---|---|
| Branded | 40% | "SubzAgency", "subzagency.online" |
| Naked URL | 25% | "https://subzagency.online" |
| Partial-match keyword | 15% | "3D web design agency", "AI automation services" |
| Generic | 10% | "click here", "visit website", "learn more" |
| Exact-match keyword | 5% | "web design Gangtok", "AI automation agency India" |
| LSI / Semantic | 5% | "interactive website creators", "chatbot integration service" |

**Red Flags — Never Pursue:**
- Link farms or PBNs with obvious footprints
- Sites with gambling, pharma, adult, or crypto-spam content
- Directories requiring reciprocal links (link exchange schemes)
- Any site offering "guaranteed DA 50+ links for $5"
- Sitewide sidebar or footer links from unrelated sites

**Target 5–8 competitors** for analysis. Suggested starting list:
1. Local web design agencies in Gangtok (find via Justdial/Google Maps)
2. Indian 3D web design agencies
3. AI automation agencies in India
4. International 3D web design studios

**Step 2: Extract Backlink Data**

For each competitor, collect:
- Total number of backlinks
- Total number of referring domains
- Top referring domains (by DA)
- Anchor text distribution
- Link types (dofollow vs nofollow ratio)
- Content types earning links (blog posts, tools, pages)
- Freshness (links gained in last 3/6/12 months)

**Step 3: Identify Link Opportunities**

For each competitor backlink, ask:
- Can I get a link from the same site? (Directory, citation, guest post opportunity)
- Is the linking content something I can create a better version of? (Skyscraper technique)
- Is the linking site relevant to SubzAgency? (If yes, add to outreach list)
